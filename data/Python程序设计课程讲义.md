# Python 程序设计课程讲义

## 第一章 Python 基础与运行方式

Python 是一种解释型、高级、通用编程语言。程序通常保存为 `.py` 文件，由 Python 解释器从上到下执行。变量不需要提前声明类型，但每个对象都有确定的类型。变量名只能由字母、数字和下划线组成，不能以数字开头，也不能使用关键字。

`print()` 用于输出，`input()` 用于读取用户输入。`input()` 的返回值始终是字符串；参与数值运算前需要使用 `int()` 或 `float()` 完成类型转换。

## 第二章 条件判断与循环

条件结构使用 `if`、`elif` 和 `else`。Python 依靠缩进标识代码块，同一代码块必须保持一致缩进。比较运算包括 `==`、`!=`、`>`、`<`、`>=` 和 `<=`，逻辑运算包括 `and`、`or` 和 `not`。

`for` 循环适合遍历序列或确定次数的重复任务，`while` 循环适合在条件成立期间持续执行。`break` 立即终止当前循环，`continue` 跳过本轮剩余语句并进入下一轮。循环条件若始终为真会形成死循环。

## 第三章 组合数据类型

列表是有序、可变序列，使用方括号创建。常用方法包括 `append()` 添加元素、`remove()` 删除指定元素、`pop()` 删除并返回元素。列表索引从 0 开始，负索引从末尾开始。切片表达式 `sequence[start:stop:step]` 不包含 stop 位置。

元组是有序、不可变序列，通常使用圆括号创建。只有一个元素的元组必须写成 `(value,)`。由于不可变，元组适合保存结构固定且不应被意外修改的数据。

字典保存键值对，键必须可哈希且唯一。集合保存不重复元素，适合成员判断、去重以及交集、并集、差集运算。

## 第四章 函数

函数使用 `def` 定义。参数将输入传递给函数，`return` 将结果返回调用处；没有显式 `return` 时函数返回 `None`。位置参数按顺序匹配，关键字参数通过名称匹配，默认参数必须放在非默认参数之后。

函数内部赋值的变量通常是局部变量。读取全局变量不需要声明，在函数内修改全局变量则需要 `global`。应优先通过参数和返回值传递数据，减少对全局状态的依赖。

## 第五章 异常与文件

可能失败的代码可以放入 `try`，使用一个或多个 `except` 处理特定异常，`else` 在未发生异常时执行，`finally` 无论是否发生异常都会执行。应捕获具体异常类型，不建议无条件吞掉所有异常。

使用 `with open(path, mode, encoding="utf-8") as file` 可以确保文件用完后自动关闭。常见模式包括 `r` 读取、`w` 覆盖写入、`a` 追加写入。处理文本文件时应明确指定编码。

## 第六章 面向对象基础

类使用 `class` 定义，实例方法的第一个参数通常命名为 `self`。`__init__` 在创建实例后完成属性初始化。继承允许子类复用父类行为，方法重写允许子类提供不同实现。封装的目标是隐藏内部细节并提供稳定接口。
