## 1. [选择题] 在Python中，关于input()函数的返回值类型，下列说法正确的是？

- A. 始终为整数
- B. 始终为字符串
- C. 始终为浮点数
- D. 根据输入内容自动判断类型

**参考答案：** B

**解析：** 根据资料，input()的返回值始终是字符串；参与数值运算前需要使用int()或float()完成类型转换。

**依据：** Python程序设计课程讲义.md · 第一章 Python 基础与运行方式 — `input()` 的返回值始终是字符串；参与数值运算前需要使用 `int()` 或 `float()` 完成类型转换。

## 2. [选择题] 下列哪个是合法的Python变量名？

- A. 2name
- B. my-var
- C. _count
- D. class

**参考答案：** C

**解析：** 资料指出变量名只能由字母、数字和下划线组成，不能以数字开头，也不能使用关键字。_count符合规则，2name以数字开头，my-var含连字符，class是关键字。

**依据：** Python程序设计课程讲义.md · 第一章 Python 基础与运行方式 — 变量名只能由字母、数字和下划线组成，不能以数字开头，也不能使用关键字。

## 3. [简答题] 请简述for循环与while循环的适用场景，并说明break和continue语句的作用。


**参考答案：** for循环适合遍历序列或确定次数的重复任务，while循环适合在条件成立期间持续执行。break立即终止当前循环，continue跳过本轮剩余语句并进入下一轮。

**解析：** 直接依据资料第二章关于循环结构的描述进行总结。

**依据：** Python程序设计课程讲义.md · 第二章 条件判断与循环 — `for` 循环适合遍历序列或确定次数的重复任务，`while` 循环适合在条件成立期间持续执行。`break` 立即终止当前循环，`continue` 跳过本轮剩余语句并进入下一轮。

## 4. [选择题] 关于Python中的元组，下列说法错误的是？

- A. 元组是有序且不可变的序列
- B. 创建只有一个元素的元组时，必须在该元素后加逗号
- C. 元组适合保存结构固定且不应被意外修改的数据
- D. 元组使用方括号创建

**参考答案：** D

**解析：** 资料明确指出元组通常使用圆括号创建，而方括号用于创建列表。元组是有序、不可变序列，单元素元组需加逗号，适合保存固定结构数据。

**依据：** Python程序设计课程讲义.md · 第三章 组合数据类型 — 元组是有序、不可变序列，通常使用圆括号创建。只有一个元素的元组必须写成 `(value,)`。由于不可变，元组适合保存结构固定且不应被意外修改的数据。

## 5. [简答题] 在Python异常处理结构中，else和finally子句分别在什么情况下执行？使用with open()打开文件有什么优势？


**参考答案：** else子句在未发生异常时执行，finally子句无论是否发生异常都会执行。使用with open()可以确保文件用完后自动关闭。

**解析：** 依据资料第五章关于异常处理机制和文件操作模式的说明。

**依据：** Python程序设计课程讲义.md · 第五章 异常与文件 — `else` 在未发生异常时执行，`finally` 无论是否发生异常都会执行。使用 `with open(path, mode, encoding="utf-8") as file` 可以确保文件用完后自动关闭。
