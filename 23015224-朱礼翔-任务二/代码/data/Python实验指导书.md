# Python 程序设计实验指导书

## 实验一 成绩等级判断

读取 0 到 100 的成绩并输出等级：90 分及以上为 A，80 至 89 为 B，70 至 79 为 C，60 至 69 为 D，低于 60 为 F。程序必须先检查输入能否转换为数字，并验证范围。非法输入应给出提示而不是直接崩溃。

## 实验二 列表统计

输入若干整数保存到列表，计算最大值、最小值、平均值和去重后的元素个数。不得直接修改原列表。测试数据至少覆盖空输入、重复值、负数和只有一个元素四种情况。

## 实验三 文件词频统计

使用 `with open(..., encoding="utf-8")` 读取文本，将英文单词统一转为小写后统计频率。结果按频率降序排列，频率相同则按单词字母顺序排列。文件不存在时捕获 `FileNotFoundError` 并提示用户检查路径。

## 实验四 学生类设计

设计 `Student` 类，包含姓名和成绩列表。提供添加成绩、计算平均分和返回等级的方法。构造方法负责初始化属性，方法之间通过返回值协作。测试时创建至少两个互不影响的实例，避免将可变列表作为默认参数。
